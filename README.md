# RMC Phonon Dynamics

A browser-based tool for calculating phonon properties from structural ensembles
generated by [RMCProfile](https://rmcprofile.ornl.gov). It loads an RMC ensemble,
computes the mass-weighted displacement-covariance phonon band structure on
**WebGPU**, animates the 3D vibrational modes, and simulates powder inelastic
neutron scattering S(|Q|,E) and the phonon density of states — entirely
client-side, with no backend server.

**▶ Live app: https://drthyang.github.io/rmc-phonon-dynamics/**

> The application lives in [`web/`](web/) (React + Vite + WebGPU). The earlier
> Python runner + viewer (`src/`, `src_gpu/`, `rmcph_gui/`, `viz/`) have been
> retired to [`archive/`](archive/) and are kept for reference only.

## Features

- **Phonon band structure** — dispersion E(k) along a high-symmetry path, with
  drag-to-zoom and soft-mode (imaginary-frequency) highlighting.
- **3D mode viewer** — click a band point to animate that mode; ball-and-stick /
  spacefill / wireframe, bonds, displacement vectors, per-element colours.
- **Simulated INS** — powder-averaged S(|Q|,E) heatmap with a kinematic cutoff,
  plus the phonon DOS.
- **Fit quality** — per-configuration Rw overview and F(Q)/G(r) overlays from
  RMCProfile output.
- **Export** — phonopy-compatible `band.yaml` / `band.json`.

## Requirements

A **Chromium-based browser** (Chrome/Edge): the app needs both **WebGPU** and the
**File System Access API** (`showDirectoryPicker`), which Firefox/Safari do not
yet provide. A secure context (the hosted HTTPS site, or `http://localhost`) is
required.

## Run locally

```bash
cd web
npm install
npm run dev          # open the printed http://localhost:5173
npm run validate     # science + UI-mapping tests
npm run build        # static build → web/dist/
```

See [`web/README.md`](web/README.md) for the in-app workflow and
[`web/FEATURE_PARITY_REPORT.md`](web/FEATURE_PARITY_REPORT.md) for how it maps to
the original Python implementation.

## Repository layout

| Path | What |
|---|---|
| [`web/`](web/) | The application (React + Vite + WebGPU). |
| [`archive/`](archive/) | Retired Python engines (`src/`, `src_gpu/`), the FastAPI GUI (`rmcph_gui/`), the standalone viewer (`viz/`), design handoffs, and historical notes. |
| `.github/workflows/` | CI that builds `web/` and deploys it to GitHub Pages. |

`data/` and `results/` are local-only (git-ignored).

## Hosting

The live site is built from `web/` and deployed to GitHub Pages by
[`.github/workflows/deploy-pages.yml`](.github/workflows/deploy-pages.yml) on
every push to `main` (Vite `base` set to the project subpath via `VITE_BASE`).
